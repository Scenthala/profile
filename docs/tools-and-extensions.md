# Recommended Tools & Extensions

## VS Code Extensions

### Required
| Extension | Purpose |
|-----------|---------|
| **ES7+ React Snippets** | React component scaffolding |
| **Vite** | Vite integration and debugging |
| **ESLint** | Code linting |
| **Prettier** | Code formatting |

### Diagrams & Documentation
| Extension | Purpose |
|-----------|---------|
| **Mermaid Preview** (`bierner.markdown-mermaid`) | Render `.mmd` diagrams in VS Code |
| **Markdown Preview Enhanced** | Rich markdown preview |
| **Draw.io Integration** (`hediet.vscode-drawio`) | Visual diagram editor |
| **PlantUML** | Alternative UML diagrams |

### Development
| Extension | Purpose |
|-----------|---------|
| **CSS Variable Autocomplete** | Autocomplete for CSS custom properties |
| **Color Highlight** | Show colors inline in code |
| **Auto Rename Tag** | Sync rename HTML/JSX tags |
| **GitLens** | Git blame, history |
| **GitHub Copilot** | AI pair programming |

### Testing & Quality
| Extension | Purpose |
|-----------|---------|
| **Vitest** | Test runner integration |
| **axe Accessibility Linter** | Accessibility checks |
| **Lighthouse** | Performance auditing |
